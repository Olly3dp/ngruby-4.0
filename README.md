# NGRUBY - Landing Page com Chatbot de IA

Landing page da NGRUBY com chatbot de IA integrado para conversão de clientes. Projeto adaptado do tutorial do CódigoPrático.

## Funcionalidades

- Chatbot Dinâmico: Fluxo de perguntas estruturado para captura de dados (Nome, Serviço, Descrição, Contato).
- Persistência com LocalStorage: Os dados não são perdidos se o usuário atualizar a página.
- Botão Flutuante Inteligente: Identifica usuários que já iniciaram atendimento e exibe saudação personalizada.
- Design Moderno: Interface com Tailwind CSS, paleta NGRUBY (amarelo mostarda, vermelho bordo, preto).
- Integração com WhatsApp: Geração automática de links com mensagens pré-definidas.

## Tecnologias

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- LocalStorage API

## Como usar

1. Clone este repositório ou baixe os arquivos
2. Abra o arquivo `index.html` no navegador
3. Para testar o chatbot, clique em "Fale Conosco"
